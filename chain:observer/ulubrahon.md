1. Spam Email Filter
•	An email arrives
•	Check for spam keywords
•	If unclear, check sender reputation
•	If still unclear, send for manual review
2. Weather Update
•	The weather changes during the day
•	Phone app, website, and display board need the update
•	Each one shows the weather in its own way
•	When the weather changes, all should update automatically
3. Discount Approval
•	A discount request is made
•	Small discounts approved by staff
•	Larger discounts go to manager
•	Big discounts go to owner
4. Chat Group Notification
•	A user sends a message in a group chat
•	All group members should receive the message
•	Members can join or leave the group
•	Only current members should be notified
5. Bug Fixing Workflow
•	A bug is reported
•	Junior developer checks first
•	If not fixed, pass to senior developer
•	Then to team lead if needed
6. Online Store Order Status
•	An order status changes
•	The customer should be notified
•	The delivery team may also be notified
•	The inventory system may update its records
7. Leave Approval
•	An employee submits a leave request
•	Supervisor can approve small leaves
•	Manager handles longer leaves
•	Director handles special cases
8. Class Attendance
•	A teacher marks attendance
•	The student record should update
•	The parent notification system may be informed
•	The adviser dashboard should update
9. Order Validation System
•	An order is placed
•	Check stock availability
•	Check payment
•	Check delivery details
10. Student Grade
•	A teacher updates a student's grade
•	The student should be notified
•	The parent may also be notified
•	The class adviser may also receive the update
